# Pizza Pursuit
Pizza Pursuit is a 2-D Platformer with 3 Levels and 3 different enemies. 

# Gameplay
- Collect bread to raise score, challenge yourself to get the highest score possible.
- There is a built in level editor where you can build and play your own level.
